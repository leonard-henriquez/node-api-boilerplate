## :package: Installation

Check you have yarn installed:

```sh
  yarn -v
```

If you have an error install yarn:
```sh
  npm install -g yarn
```

Install dependencies:

```sh
  yarn install
```

Setup environment variables:

```
  cp .env.example .env
```

## :rocket: Usage

Start server:

```sh
  yarn start
```
